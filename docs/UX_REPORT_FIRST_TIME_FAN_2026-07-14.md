# 使用体验报告 1：第一次进入的球迷

## 用户目标

十秒内知道谁在比赛、当前状态是什么、接下来能做什么。

## 体验结果

- 首屏把比赛、比分挑战和比赛剧场放在主路径，赛程、球队和设置留在次级导航。
- “比分挑战”使用 1,000 本地积分并明确无现金价值；输入、锁定和修改规则在操作点附近可见。
- 赛前、延迟、直播和回放有不同文案，用户不会把已完成归档误认成当前直播。
- AI 三模式分别回答“发生了什么”“为什么重要”“我错过了什么”，比单句状态更适合第一次进入。
- 球员来源只有内部编号时不显示编号或伪姓名，避免新用户误解。

## 主要摩擦复检

- 已删除主路径中的 endpoint、fixture ID、source team、next signal 等术语。
- 已删除重复的官方数据标签和移动端输入框外的字符计数。
- 390px 下球队全名、比分、场上人数和主要按钮均可读。

## 结论

第一次进入不需要教程即可完成“看懂比赛 -> 参与挑战 -> 看关键事件/补课”。本地体验通过。
