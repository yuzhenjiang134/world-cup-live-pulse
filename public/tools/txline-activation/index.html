Safe TxLINE Activation

This page never asks for a private key or seed phrase. It uses your browser wallet's signMessage method to sign the official TxLINE activation message, then calls the official activation endpoint.

Do not paste private keys here

If a private key was shared anywhere, treat that wallet as compromised. Use a new clean wallet for hackathon testing. Only public wallet addresses, transaction signatures, guest JWTs, and final API tokens belong in this workflow.

Before you start

Keep the same network for the Solana subscription transaction, guest JWT, and TxLINE API host. Mainnet free tiers include Service Level 1 delayed data and Service Level 12 real-time data; devnet availability depends on the official tier table.
