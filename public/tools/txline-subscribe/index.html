Subscribe First, Then Activate

This page performs the missing official step before API activation: it asks your browser wallet to send TxLINE's on-chain subscribe instruction, waits for the transaction signature, and then sends you to the safe activation helper.

Wallet safety boundary

This helper never asks for a private key or seed phrase. It only requests a normal wallet transaction approval. Review the wallet popup carefully: the program id, network, and free tier service level must match the TxLINE docs.

Official references

TxLINE documents that API access requires a free-tier on-chain subscription first. Keep the same network for Solana RPC, TxLINE program id, guest JWT, and activation endpoint.
