## Why one English cut

One cut keeps the submission unambiguous. It preserves the fan-first hierarchy while mapping the working product to fan accessibility, real-time responsiveness, originality, commercial path, and completeness. It gives TxLINE scope and final-score rules enough time without exposing credentials or becoming a developer walkthrough.

## Reproducible generation

The scene manifest lives in `scripts/record-demo-video.mjs`. The current screenshots are generated by the final browser E2E and copied to the ignored `demo-assets/screenshots/` directory.

```powershell
python scripts\render-demo-video.py --variant B
& .\scripts\render-demo-narration.ps1 -Variant B
```

`npm run demo:video` performs the same two steps. The renderer uses the local FFmpeg encoder. When `WCLP_VOICE_PROFILE_ID` is set locally, the authorized local voice service receives per-scene English emotion direction; the profile ID is never written into source, output captions, or the public bundle.
